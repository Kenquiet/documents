# Mouseover (鼠标移至) 与 Mouseenter 的区别

Mouseover事件：不论鼠标指针被选元素或其子元素，都会被被触发事件。
Mouseenter事件：只有在鼠标指针穿过被选元素时，才会触发
所以以后我们在注册鼠标经过事件时使用mouseenter(鼠标移进)和mouseleave(鼠标移出)
