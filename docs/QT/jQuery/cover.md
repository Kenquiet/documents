## 入口函数
### 与js 入口函数比较
1. js的入口函数执行要比jQuery 的入口函数执行得晚一些
2. jq 的入口函数会等待页面加载完成才执行，但是不会等图片加载完成
3. js 的入口函数会等待页面加载完成，而且还会等待图片的加载完成才开始执行

### jq的入口函数
1. jq 有两种入口函数的写法
    ```javascript
    $(document).ready(function () {
           console.log("这是jQuery入口函数的第一种写法")
        });
        $(function () {
            console.log("这是jQuery入口函数的第二种写法")
        })
    ```
